API Changes (matching docs/pakasir.md):

BEFORE (wrong):
  - POST /api/checkstatus (doesn't exist)
  - checkout_url: {domain}/{slug}/{order_id} (missing /pay/ and amount)

AFTER (correct):
  - GET /api/transactiondetail?project=...&amount=...&order_id=...&api_key=...
  - checkout_url: {domain}/pay/{slug}/{amount}?order_id=...&qris_only=1

Webhook Payload (official format):

    "amount": 22000,
    "order_id": "tg12345-ORDER123",
    "project": "your-project",
    "status": "completed",
    "payment_method": "qris",
    "completed_at": "2025-09-10T08:07:02.819+07:00",
    "metadata": {
      "telegram_id": 12345,
      "telegram_username": "user123"
    }
  }
